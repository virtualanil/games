# Typing Game 

A 3D visually stunning typing shooter game built with Canvas API and Firebase.

## Setup Instructions
1. This game uses ES6 modules. To run it locally, you **must** use a local web server (e.g., Live Server in VS Code) because of CORS restrictions on fetching `data/words.json` and module imports.
2. Open `index.html` via your local server.

## Folder Structure Maintained
* `css/`: Separated styles for layout, components, and animations.
* `js/`: Modularized logic (Game, Player, Enemy, Audio, Firebase).
* `data/`: JSON files for dynamic configuration.
* `assets/`: Add your external assets here.
